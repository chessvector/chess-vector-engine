3/// UCI chess engine binary
4///
5/// This binary provides a UCI (Universal Chess Interface) compliant chess engine
6/// that can be used with any UCI-compatible chess GUI such as:
7/// - Arena
8/// - ChessBase
9/// - Fritz
10/// - Scid vs. PC
11/// - BanksiaGUI
12/// - And many others
13///
14/// Usage:
15/// 1. Compile: `cargo build --release --bin uci_engine`
16/// 2. Add the resulting binary to your chess GUI as a new engine
17/// 3. The engine will communicate using standard UCI protocol
18///
19/// Features:
20/// - Hybrid evaluation combining pattern recognition and tactical search
21/// - GPU acceleration when available (CUDA/Metal) with CPU fallback
22/// - Opening book integration
23/// - Configurable search depth and time management
24/// - Advanced pattern-based move recommendations
25/// - LSH-accelerated similarity search
26///
27/// UCI Options:
28/// - Hash: Hash table size in MB (1-2048, default 128)
29/// - Threads: Number of search threads (1-64, default 1)
30/// - MultiPV: Number of principal variations (1-10, default 1)
31/// - Pattern_Weight: Weight for pattern evaluation vs tactical (0-100, default 60)
32/// - Tactical_Depth: Maximum tactical search depth (1-10, default 3)
33/// - Pattern_Confidence_Threshold: Minimum confidence for pattern-only evaluation (0-100, default 75)
34/// - Enable_LSH: Use LSH acceleration for similarity search (true/false, default true)
35/// - Enable_GPU: Use GPU acceleration when available (true/false, default true)

37    run_uci_engine();
